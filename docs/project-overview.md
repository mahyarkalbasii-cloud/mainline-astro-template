# Project Overview

## Project
PersianSaze marketing website rebuild on top of the Mainline Astro template.

## Primary goal
Explain clearly what PersianSaze does and convert relevant visitors into leads.

## Business role of the site
This site is separate from the product application. It is a marketing website, not the app itself.

## What the site must achieve
- Make the value proposition understandable in a few seconds
- Show who the product is for
- Explain how the system works
- Present plans clearly
- Support SEO and AEO
- Drive demo/contact actions

## Core audience
Suppliers of construction products and services who sell to projects under construction.

## Geographic scope
Tehran, Karaj, and Lavasan.

## Primary conversion
Request demo / contact / lead submission.

## Secondary conversion
Read more about how the product works, plans, and use cases.

## Positioning
PersianSaze is not just a contact database, a simple CRM, or an SMS tool. It is a system for helping suppliers find the right construction projects at the right time and move sales forward in a more targeted way.

## Core value pillars
- Live project data
- Execution tools
- Training
- AI support

## Website scope for V1
- Home
- How it works
- Pricing
- FAQ
- Contact
- About
- Blog

## Constraints
- Use the existing Mainline Astro template as the base
- Prefer adapting existing components over building new ones
- Keep the site light, clear, and fast
- Do not overcomplicate interactions
- Preserve content accuracy
- The source content must not be paraphrased where exact transfer is required

## Success criteria
- Clear homepage messaging
- Clean site structure
- Strong SEO/AEO foundation
- Ready for iterative Codex execution
- Easy to customize without breaking the template system
